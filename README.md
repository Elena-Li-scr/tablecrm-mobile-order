# TableCRM Mobile Order

Мобильная форма создания заказа через API TableCRM.

Проект выполнен как тестовое задание на позицию React Frontend Developer.

## 🚀 Стек

- React
- TypeScript
- Vite
- shadcn/ui
- Axios
- React Hook Form
- Zod

---

## 📱 Функциональность

### Авторизация
- Ввод API token
- Сохранение токена в localStorage

### Работа с API
Получение данных из TableCRM API:

- Клиенты
- Склады
- Счета
- Организации
- Типы цен
- Товары

### Создание продажи
- Поиск клиента по телефону
- Выбор организации
- Выбор склада
- Выбор счета
- Выбор типа цены
- Добавление товаров
- Изменение количества товаров
- Расчет итоговой суммы

### Отправка продажи
- Создать продажу
- Создать и провести продажу
